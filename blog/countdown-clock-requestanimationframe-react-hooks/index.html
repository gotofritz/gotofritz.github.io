A countdown clock using requestAnimationFrame with React hooks

see https://www.youtube.com/watch?v=XHVK-CAysZ0&feature=youtu.be For giggles I decided to compare requestAnimationFrame with React hooks and vanilla JS, so I came up with the idea of a simple countdown clock. This is the React hooks version; you can find the vanilla version here

Using requestAnimationFrame can be a little tricky with React hooks. A countdown clock is a simple way to get familiar with how it works

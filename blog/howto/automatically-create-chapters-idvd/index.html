Automatically Creating Chapters in iDVD

2012-04-27, Fri
HowTo
Comments

If you have a large movie file you want to burn to DVD with iDVD, and would like to split it into chapters automatically, here’s what to do.

  1. create a new Magic DVD project (File > New… > Magic DVD

  2. drag the movie file(s) into it

  3. click on the ‘Create Project’ button, bottom right

  4. select the Chapters theme from the list

  5. select the movie file itself in the tree view

  6. Select Advanced > Create Chapter Markers

  7. Enter how many minutes long you want the chapters to be

  8. Adjust other settings as required

  9. Burn DVD

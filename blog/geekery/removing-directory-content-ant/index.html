Removing directory content with Apache Ant

2012-08-06
ant
This post is too old to be of any use. It has been archived.

I always trip over this, even though it should be simple.

How to empty a directory, without deleting the directory itself, using Apache Ant.

<delete
  includeemptydirs = "true"
  ><fileset
      dir      = "${the_dir}"
      includes = "**/*"
    />
</delete>

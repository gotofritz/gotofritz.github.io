Social Media

Asos hired me to help them kickstart their social media presence. I built a few Facebook / Bebo / MySpace apps for them, as well as tools to speed up their publishing workflow to these platforms.

Some of the projects while at CMW London involved a social media element - Mess With Your Head was a module within Bebo, the social networking site targeted at teenagers.

We also built a Bebo game as part of the Creme Egg online campaign.

New Media Knowledge is a collaborative media publishing site for the digital industry.

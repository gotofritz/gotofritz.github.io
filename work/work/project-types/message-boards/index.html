Message Boards

In the early days of the web it was a given that a website had a message board. It didn’t take long for agencies to cotton on to the fact that message boards need moderation, and that costs money. These days dedicated social networking sites fulfill the same function.

A few of the early sites I worked on included message boards, which I coded in Perl and CGI. New Media Knowledge is a collaborative media site for which I built the front-end.

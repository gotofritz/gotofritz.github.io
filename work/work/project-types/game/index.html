Game

That is game as in video games, as opposed to a simple competition.

E.ON Great Saves included a grid based spot the save game built in Flash and SOAP.

Quacky Races was a tamagotchi for O2.

The Littlewoods Island Game was a simple quiz.

The pre-Twisted, Creme Egg website included a Flash game, which was also developed as a downloadable mobile application. We also created a game for Bebo, a social network. Creme Egg Twisted also had a game, which was outsourced.

The Cooling Gel game for Nivea was a simple but sweet game of balance, built in Flash.

For Hitchhikers’ Guide to the Galaxy we created 42 (of course) fast mini games based on 6 templates.

Play More included 15 or so zany little games.

Barmer included a Shockwave tamagotchi clone, based around an orphan rather than a pet. A cunning way to get lots of user data, with the excuse of having to fill in adoption forms.

Henkell and Wodka Gorbatschow had one game each, both built in Shockwave - a car racer and a mouse tracer.

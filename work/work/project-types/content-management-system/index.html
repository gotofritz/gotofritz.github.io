Content Management System

I always try to separate content from presentation, and try to use tools to make the content easily manageable. Even if it’s just XML files being fed to a Flash interface.

E.ON Great Saves / Talking Energy are two complex websites built on the same CMS platform, eZpublish.

The E.ON Youtube Channel uses a simple custom built CodeIgniter CMS to upload videos to YouTube and add extra attributes to them.

This portfolio is built with Wordpress. It includes a few complex custom plugins.

Freesim is a complex campaign management and reporting system.

I built simple management tools to allow content owners to edit XML sources files for ASOS magazine, LittleWoods Style Advisor, and KRZR.

Both the LCR and London Shard sites have a content managed news section with gallery.

For the IMO project we designed a reasonably complex CMS from scratch.

The other projects were either functional amends to existing CMS systems (BeCrypt, Foyer), or designing a front end architecture to integrate with an existing CMS (Levi EU, NMK, Bet2Go).
